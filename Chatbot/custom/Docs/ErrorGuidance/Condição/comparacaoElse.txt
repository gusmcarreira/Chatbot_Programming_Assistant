Checkpoint:A grande diferença do if/elif para com o else, está que no else não se usa uma comparação, por exemplo:

<code>
x = 0
if x == 1:
  print("X é igual a 1!")
else:
  print("X é igual a 0!")
</code>

<code_print>X é igual a 0!

Pense que o if é um "se", o elif é outro "se", enquanto que o else é um "resto", ou seja se nenhum das outras condições é verdade então corre o que está declarado no else.

Veja se no seu algoritmo segue essa sintaxe!<end>
