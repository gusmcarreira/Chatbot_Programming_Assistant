Pergunta:O seu erro ser numa condição. Vou-lhe propor um desafio! Dado o código:

<code>x = 0

if:
  print("X é igual a 0!")
else:
  print("X não é igual a 0!")</code>

Que é suposto gerar a seguinte saida:

<code_print>X é igual a 0!

Sabe dizer o que está em falta?<end>
----------------------------------------------------------------------------------<end>
Resposta:x == 0<sep>x==0<sep>comparação<end>
----------------------------------------------------------------------------------<end>
Explicação:A resposta correta é a comparação, ou mais especificamente, x == 0.<end>
----------------------------------------------------------------------------------<end>
-###-
----------------------------------------------------------------------------------<end>
Checkpoint:Ao contrário da instrução else, quando usamos uma instrução if ou elif, necessitamos incluir uma comparação, no exemplo acima o correto seria "if x == 0:".

Veja agora no seu código se quando usou um if e/ou elif sempre usou uma comparação!<end>
