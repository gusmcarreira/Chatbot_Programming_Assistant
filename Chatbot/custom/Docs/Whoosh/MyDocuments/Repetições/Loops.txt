Repetição (loop, ou laços de repetição) é a repetição de um comando, ou conjunto de instruções, até que uma dada condição seja satisfeita.

Exemplo:

<code>
print("Números: ")

for numero in [1, 1, 2]:
  print(numero)
</code>

<code_print>Números:
1
1
2

Existem dois tipos de repetições (loops): while e for.

Sugestões:Instrução while<sep>Instrução for<sep>Escopo de uma repetição
-#-
A instrução while é um tipo de repetição em que um comando/s será executado ENQUANTO a condição for verdadeira.

A sua declaração segue a sintaxe:

<code_print>while condição:

Exemplo:

<code>
i = 0

while i < 3:
  print(i)
  i = i + 1

<code_print>0
1
2

É importante notar a condição i < 6, seguida por pois pontos (:), e depois o bloco de código indentado (espaçado) que determina o escopo da instrução while (conjunto de instruções a serem realizados enquanto a condição é verdadeira).

Este contém dois comando, um que imprime o numero em i, e outro que vai adicionando 1 ao i, pois caso contrário a instrução nunca pararia de correr, pois i seria sempre menor que 3.

Nota: o número 3 não é imprimido pois a condição só é verdade enquanto i for MENOR que 3, para o 3 ser impresso a condição teria de ser MENOR OU IGUAL (<=).

Sugestões:Instrução for<sep>Escopo de uma repetição
-#-
A instrução for é a mais comum instrução de repetição, que é usado com objetos iteráveis tais como listas e intervalos.

A sua sintaxe é:

<code_print>for var_de_indexação in objeto_a_iterar:

Exemplo:

<code>
for i in range(3):
  print(i)
</code>

<code_print>0
1
2

A função range é como que uma lista de números começando em 0. No exemplo acima é então uma lista de 3 números (0, 1 e 2).

Outro exemplo:

<code>
cores = [azul, branco]
for cor in cores:
  print(cor)
</code>

<code_print>azul
branco
amarelo

Observação: Pode tentar ler como "por cada cor em cores", sendo que "cor" poderia ter qualquer outro nome de variável, por exemplo, i (como no primeiro exemplo), mas sempre ajuda ter um nome com significado.

Sugestões:Instrução while<sep>Escopo de uma repetição
-#-
O escopo de uma repetição é o bloco de código que será executado quando a sua condição se verifica.

Esse escopo é determinado pelo número de espaços (indentação) que vem depois da sua declaração, tendo de ser no MINIMO 1, no entanto, tem de ser SEMPRE O MESMO NÚMERO para os diferentes blocos de código.

Exemplo:

<code>
num1 = 0

while num1 < 3:
| print("Menor que 3")
| print("Escopo correto")
| num1 = num1 + 1

while num1 < 3:
| print("Menor que 3")
| #Número de espaços
| #diferente (erro)
|  print("Escopo incorreto")
|  num1 = num1 + 1
</code>

Os escopos são portanto controlados pelo número de espaços. Sendo que uma repetição pode por ela conter mais repetições, ou condições que contêm também o seu escopo.

<code>
num1 = 0

while num1 < 6:
| print("Menor que 6")
| if num1 < 2:
| | #Escopo confição
| | print("Ainda menor que 2")
| num1 = num1 + 1
</code>

Nota: o simbolo (|), está presente meramente para demonstração dos diferentes escopos (indentações).
