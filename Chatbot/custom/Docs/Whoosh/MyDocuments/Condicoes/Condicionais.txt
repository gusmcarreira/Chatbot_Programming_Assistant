Precisamos de condicionais, pois os programas estão a todo o momento analisando condições que determinam o seu funcionamento. Por exemplo, o acesso a uma conta no Instagram depende de uma condição: o e-mail e senha informados devem ser iguais ao cadastro em seu banco de dados. Da mesma forma, o levantamento de dinheiro num multibanco passa por uma condição: o valor desejado deve ser menor ou igual ao saldo da conta. Em outras palavras, só pode levantar o que possui. 

Como pode-se perceber, as condições são parte fundamental de qualquer programa. Na programação utilizamos esse recurso para realizar verificações matemáticas entre variáveis. No exemplo do banco, comparamos se a variável que guarda o valor solicitado para saque é **menor ou igual** ao valor guardado na variável que armazena o saldo bancário.

-#-

As operações de comparação suportadas são:
> (maior que)
< (menor que)
== (igualdade)
>= (maior ou igual)
<= (menor ou igual)

-#- 

A sintaxe de uma condição (if statement) em Python é:

	if variável operador variável:

Constituído pela palavra if, duas variáveis para comparação, o operador dessa comparação, e : (dois pontos). As verificações de uma condição SEMPRE resultam em verdadeiro ou falso. A escolha entre um ou outro dependerá do valor das variáveis.

Exemplo:

	saldo = 100
	levantamento = 50

	if saldo >= levantamento:
		print("Possível levantar dinheiro")

	Possível levantar dinheiro

A comparação acima resultará em verdadeiro, pois o saldo é maior que o levantamento. Quando essa situação ocorre, o conjunto de instruções que pertencem à condição serão executadas. 

Observe como a instrução "print" apresenta um espaçamento (indentação) no inicio da linha. Isso é SEMPRE NECESSÁRIO e define quais as instruções serão executadas quando a comparação for verdadeira. 

-#-

O uso do if não resolve todos os nosso problemas, pois quando a comparação de uma condição não for verdadeira também pode ser preciso realizar alguma ação. Em casos como esse podemos utilizar uma instrução conhecida por "else".

Exemplo:

	saldo = 12
	levantamento = 35

	if saldo >= levantamento:
    		print("Levantamento realizado com sucesso!")
	else:
    		print("Não foi possível realizar o levantamento!")

	Não foi possível realizar o levantamento!

Há dois detalhes para ficar atento no uso do else. O primeiro é que não é preciso escrever um comparação e isto faz sentido, pois o seu escopo SEMPRE será executado quando a comparação do if resultar em falso. O segundo ponto, é que, assim como no if, é preciso escrever as instruções do seu escopo com o espaçamento (indentação) e usar os : (dois pontos).

-#-

Poderá também ser necessário várias condições, ou seja, condições em cadeia (um após o outro). Em casos como esse podemos utilizar uma instrução conhecida como "elif".

Exemplo

	saldo = 100
	levantamento = 100

	if saldo > levantamento:
    		print("Levantamento realizado com sucesso!")
	elif saldo == levantamento:
    		print("Levantamento realizado com sucesso! Conta ficou a 0")
	else:
    		print("Não foi possível realizar o levantamento!")

	Levantamento realizado com sucesso! Conta ficou a 0

Não está limitado a uma única instrução elif, e tal como a condição if, esta necessita de uma condição, seguida por : (dois pontos). Observe como a instrução "print" apresenta um espaçamento (indentação) no inicio da linha. Isso é SEMPRE NECESSÁRIO e define quais as instruções serão executadas quando a comparação for verdadeira. 

-#-
A instrução "else" é usada quando nenhuma das outras condições resultou num verdadeiro (True).

Exemplo:

	saldo = 10
	levantamento = 100

	if saldo > levantamento:
    		print("Levantamento realizado com sucesso!")
	elif saldo == levantamento:
    		print("Levantamento realizado com sucesso! Conta ficou a 0")
	else:
    		print("Não foi possível realizar o levantamento!")

	Não foi possível realizar o levantamento!
