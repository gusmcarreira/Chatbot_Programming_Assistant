A linguagem de programação Python pode ser usada:
   Num servidor para criar aplicativos web.
   Para comunicar com bases de dados.
   Para ler e modificar ficheiros,
   Para lidar com big data e realizar matemática complexa.

Python porque ocupa hoje uma posição única no cenário de linguagens de programação. É uma linguagem simples e fácil de aprender.

Com Python, pelo fato de a linguagem ser simples, o aprendizado dos conceitos de programação e algoritmos assumem o papel principal que lhes é devido na trajetória de aprendizado de alguém.

Python possui também um número gigante de bibliotecas disponíveis, o que facilita muito a criação de programas na linguagem. Para muitas tarefas, você encontrará uma biblioteca pronta que facilitará muito sua vida. Com isso, você não precisa ficar reinventando a roda toda vez que quiser escrever um programa para resolver um problema específico.

Esta linguagem é atualmente a linguagem mais usada em Ciência dos Dados, Aprendizado de Máquina, e Inteligência Artificial.
