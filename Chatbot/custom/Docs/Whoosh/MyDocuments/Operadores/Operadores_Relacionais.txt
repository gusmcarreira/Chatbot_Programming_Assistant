Python também possui operadores de comparação como os da matemática (<, >, <=, >=, ==, !=)

<code>
==

True se a e b são IGUAIS
5 == 2 resulta em False
</code>

<code>
!=

True se a e b são DIFERENTES
5 != 2 resulta em True
</code>

<code>
>

True se a é MAIOR que b
5 > 2 resulta em True
</code>

<code>
<

True se a é MENOR que b
5 < 2 resulta em False
</code>

<code>
>=

True se a é MAIOR OU IGUAL a b
5 >= 2 resulta em True
</code>

<code>
<=

True se a é MENOR OU IGUAL a b
5 <= 2 resulta em False
</code>

Tome cuidado: não confunda o operador == (teste de igualdade) com o operador = (operador de atribuição). Este último é usado para atribuir um valor a uma variável. É muito comum as pessoas confundirem esses operadores, principalmente em condicionais (if e else).
