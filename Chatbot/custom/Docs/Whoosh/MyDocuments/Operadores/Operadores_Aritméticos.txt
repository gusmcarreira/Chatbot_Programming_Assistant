Além de armazenarem dados, um dos grandes ganhos no uso de variáveis é poder realizar operações em seus dados.

Exemplo:

<code>
x = 2
y = 3
resultado = x + y
</code>

Onde na primeira linha nós criamos uma variável de nome x e guardamos o valor 2, na segunda linha criamos uma variável de valor 3, e por fim, criamos uma variável resultado que guardará o resultado da operação matemática de soma de x com y (5).

Em python existem algumas particularidade na hora de realizar operações matemáticas, por vezes não são utilizados os mesmos simbolos que estamos acostumados:

<code>
Operador matemárico: Soma
Símbolo da matemática: +
Simbolo em programação: +
</code>

<code>
Operador matemárico: Subtração
Símbolo da matemática: -
Simbolo em programação: -
</code>

<code>
Operador matemárico: Multiplicação
Símbolo da matemática: x
Simbolo em programação: *
</code>

<code>
Operador matemárico: Divisão
Símbolo da matemática: ÷
Simbolo em programação: /
</code>

<code>
Operador matemárico: Potência
Símbolo da matemática:
Simbolo em programação: **
</code>

<code>
Operador matemárico: Raiz
Símbolo da matemática:
Simbolo em programação: variavel**.5
</code>

Sugestões:Como se faz a soma?<sep>Como se faz a subtração?<sep>Como se faz a divisão?<sep>Como se faz a multiplicação?
-#-
A soma é feita com o operador +

Exemplo:

<code>
numero1 = 10
numero2 = 2.5
resultado = numero1 + numero2
	
print(resultado)
</code>

<code_print>12.5
-#-
Para subtrair usa-se o operador -

Exemplo:
	
<code>
numero1 = 10
numero2 = 8.5
resultado = numero1 - numero2
	
print(resultado)
</code>

<code_print>1.5
-#-
Para multiplicar usa-se o operador *

Exemplo:

<code>
numero1 = 5
numero2 = 2
resultado = numero1 * numero2
	
print(resultado)
</code>

<code_print>10
-#-
Para dividir, diferente do nosso dia-a-dia, usa-se o operador /

Exemplo:

<code>
numero1 = 5
numero2 = 2
resultado = numero1 / numero2
	
print(resultado)
</code>

<code_print>2.5
-#-
A divisão ARREDONDANDO PARA BAIXO é feita com o operador //

Exemplo:

<code>
numero1 = 5
numero2 = 2
resultado = numero1 // numero2
	
print(resultado)
</code>

<code_print>2
-#-
O resto da divisão é obtido com o operador %

Exemplo:
	
<code>
numero1 = 5
numero2 = 2
resultado = numero1 ** numero2
	
print(resultado)
</code>

<code_print>1
-#-
A exponenciação é feita com o operador **

Exemplo:

<code>
numero1 = 5
numero2 = 2
resultado = numero1 ** numero2
	
print(resultado)
</code>

<code_print>25
