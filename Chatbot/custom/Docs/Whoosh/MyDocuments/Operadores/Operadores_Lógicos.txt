Em Python os operadores lógicos mais importantes são:

<table>
 	 <tr>
    		<th>and</th>
   	 	<th>"e" lógico</th>
   		<th>
			True and True resulta em True\n
			True and False resulta em False\n
			False and True resulta em False\n
			False and False resulta em False
		</th>
  	</tr>
	<tr>
   		<th>or</th>
   		<th>"ou" lógico</th>
   		<th>
			True or False resulta em True\n
			True or False resulta em True\n
			False or True resulta em True\n
			False or False resulta em False
		</th>
  	</tr>
	<tr>
    		<th>not</th>
    		<th>"não" lógico</th>
    		<th>
			not False resulta em True\n
			not True resulta em False
		</th>
  	</tr>
</table>
