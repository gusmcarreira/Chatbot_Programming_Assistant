Dado o seguinte códiog:

<code>
nome = "Leonardo"
saudacao = "Olá " + nome
</code>

No exemplo acima (vamos supor que criamos um programa para saudar um usuário) não importa quantas vezes o código seja executado o valor da variável saudacao sempre será "Olá Leonardo". Ora, o que queremos na realidade é que o usuário seja capaz de informar o seu próprio nome (entrada).

No entanto, no mundo real os valores de alguns dados são informados pelo usuário e não pelo programador.

Assim, no exemplo acima temos uma entrada que será o nome que o usuário deverá digitar. Mas, como dizemos ao Python que queremos que o usuário digite um dado? Utilizaremos a instrução input():

Exemplo:

<code>
nome = input()
saudacao = "Olá " + nome
</code>

Ao executar a primeira linha desse código, o Python aguardará até que o usuário digite algo no teclado e pressione a tecla ENTER. Apenas após isso ocorrer é que a segunda linha será executada e a concatenação realizada. Isso faz sentido, pois a variável nome guardará o que o usuário digitou no teclado, e somente de posse desse dado é que a concatenação deve ser realizada.

Nota: a função input() sempre lê strings. Por isso, caso você deseje ler um número, deverá converter o dado retornado por essa função para o formato numérico apropriado.

No entanto, ao executar o algoritmo anterior, nosso usuário após digitar seu nome no teclado não visualizará nada em sua tela.

No exemplo:

<code>
nome = input()
saudacao = "Olá " + nome
</code>

O usuário não visualizará nada em sua tela, pois as variáveis não são apresentadas na tela, elas existem apenas no código.

O que precisamos fazer é fazer com que o valor guardado na variável seja repassado para a interface do usuário (seu monitor). Para isso utilizaremos uma instrução conhecida por print().

Exemplo:

<code>
nome = input()
saudacao = "Olá " + nome
print(saudacao)
</code>

Observe que dentro do parêntesis do print nós escrevemos o nome da variável que queremos que seu valor seja apresentado na tela. Assim, após a execução deste código e escrever o nome Davi, o usuário visualizará na tela: Olá Davi.

Esse é o papel do print, indicar qual informação será a saída do nosso algoritmo.

Podemos também usar o print para apresentar uma string ou número diretamente:

<code>
print("Gostei desse livro")
print(2)
</code>

Nota: Números e strings não podem ser apresentados num mesmo print, para tal é necessário converte o número em string:

<code>
print("Meu número preferido é " + str(2))
</code>
