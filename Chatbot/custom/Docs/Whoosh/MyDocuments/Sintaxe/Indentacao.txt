A indentação refere-se ao espaço/espaços no início de cada linha de código.

-#-

Noutras linguagens de programação a indentação serve apenas para legibilidade. Em Python a indentação é bastante importante, sendo que serve para separar blocos de código.

Exemplo:

Se em vez de escrever:

    if 5 > 2:
        print("5 é maior que 2!")

Escrever:

    if 5 > 2:
    print("5 é maior que 2!")

Python retorna um erro.

-#-

O número de espaços para indentação tem de ser no MINIMO 1, no entanto, tem de ser sempre o mesmo número para cada bloco de código.

Exemplo:

O seguinte exemplo está correto, pois apesar de serem usado números de espaços diferentes, estes referem-se a blocos de código diferentes.

    if 5 > 2:
        print("5 é maior que 2!")
    if 5 > 2:
            print("5 é maior que 2!")

No entanto, o exemplo abaixo já resulta num erro de sintaxe.

    if 5 > 2:
        print("5 é maior que 2!")
            print("5 é maior que 2!")
