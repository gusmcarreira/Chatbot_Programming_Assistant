Este erro é facil de detetar:

<code>
1  # INCORRETO
2  # duas == (igualdades)
2  minha_var == 1
3
4  # CORRETO
5  # uma = (igualdade)
5  minha_var = 1
</code>

Por vezes por distração ou confusão com as condições, tentamos atribuir um valor a uma variável com = (duas igualdades). A forma correta para não cometer erro é usar apenas uma = (igualdade).

Normalmente o erro será destacado a vermelho. Se isso não aconteceu procure no seu código pela declaração de uma variável com duas == (igualades) ao invés de apenas uma..
