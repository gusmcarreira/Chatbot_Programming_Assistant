Variáveis do tipo booleano são úteis para representar dados que assumem apenas dois valores.

Pense em uma lâmpada, ela está acesa ou apagada. Você está com ou sem Internet. Em casos como esses, nós utilizamos os dados True ou False para representar a informação. Por exemplo, a lâmpada está acesa (True) ou apagada (False):

Exemplo:

<code>
1 lampada_acesa = True
2 conectado_internet = False
</code>

Vejamos agora exemplos de operações lógicas (booleanas):

<code>
1 a = 4 > 0
2 print(a)

True
</code>

O resultado vem da comparação (4 > 0) que resulta em True (verdadeiro), pois 4 é maior que 0. Este dado é depois armazenado na nossa variável "a".
