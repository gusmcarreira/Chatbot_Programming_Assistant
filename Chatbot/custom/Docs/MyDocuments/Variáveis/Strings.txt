Strings são sequências (cadeias) de caracteres ou, de forma simples, textos. Noutras palavras, uma string é simplesmente uma sequência de zero ou mais letras juntas.

Exemplo:

<code>
string_vazia = ''
uma_letra = "a"
frase = 'sou um chatbot'
</code>

Strings podem ser declaradas usando aspas simples ou duplas, o que é sempre necessário, se não for dessa forma o Python não compreenderá que você está armazenando um texto.

-#-
Para converter um dado para string usa-se a função str().

Exemplo:

<code>
# Converte inteiro para string
x = 2
s = str(x)
print(s)
</code>

Sugestões:Como obter o tipo de um dado?
-#-
Em geral, não é possível executar operações matemáticas com strings, mesmo se elas parecerem números, então coisas assim são ilegais:

<code_print>'2'-'1'
'ovos'/'fácil'
'third'*'a charm'

Mas há duas exceções, + e *.

O operador + executa uma concatenação de strings, ou seja, une as strings pelas extremidades. Por exemplo:

<code_print>
>>> primeira = 'Olá'
>>> segunda = 'Mundo'
>>> terceira = primeira + segunda
>>> terceira
OláMundo

O operador * também funciona em strings; ele executa a repetição. Por exemplo, 'Spam'*3 é 'SpamSpamSpam'. Se um dos valores for uma string, o outro tem de ser um número inteiro.
