Python possui vários tipos básicos ou fundamentais de variáveis: números inteiros (int, e.g., 2; 3; e 4), números de ponto flutuante (float, e.g., 5.2), textos (strings, e.g., "Olá") e informações booleanas (bool, True (Verdadeiro) e False (Falso))

O tipo de uma variável é dado pela função type().

Exemplo:

<code>
x = 5
print(type(5))
</code>

<code_print> <class 'int'>
