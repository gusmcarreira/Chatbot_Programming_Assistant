Int, ou inteiro, é um número inteiro, positivo ou negativo, SEM DÉCIMAS, de tamanho ilimitado.

Exemplo:

<code>x = 20</code>

Para converter uma string ou uma float para um inteiro usa-se a função int().

Exemplo:

<code>
# Converte float para inteiro
x = 2.5
s = int(x)
print(s)
