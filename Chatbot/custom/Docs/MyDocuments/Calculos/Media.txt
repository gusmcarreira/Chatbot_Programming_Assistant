<text>A média é calculada somando-se todos os valores de um conjunto de dados e dividindo-se pelo número de elementos deste conjunto.

Fórmula:
media = (x1 + x2 + x3 + ... + x3) / n

Sendo,
- x1, x2, x3,..., xn: valores dos dados
- n: número de elementos do conjunto de dados</text>

Exemplo:

<code_print>1  numero1 = 2
2  numero2 = 6
3  numero3 = 10
4
5  media = (numero1 + numero2 + numero3) / 3
6
7  print(media)

>> 6</code_print>
