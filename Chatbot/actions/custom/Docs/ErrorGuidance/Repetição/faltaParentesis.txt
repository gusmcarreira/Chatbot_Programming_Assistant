<code>
x = 0

while x < (4 + 1 :
  print(x)
  x += 1
</code>

Que é suposto gerar a seguinte saida:

<code_print>
0
1
2
3
4

Qual é o detalhe que está em falta? Um...|end|
Resposta:Se disse um parêtesis, então acertou.|end|
